# Keyboard Shortcuts

레이블링 페이지 (`#/label/:pageId`)에서만 활성화된다. INPUT, TEXTAREA, SELECT 요소에 포커스가 있을 때는 무시된다.

## 도구 모드

| 키 | 동작 |
|----|------|
| `S` | 선택 모드 (select) |
| `D` | 그리기 모드 (draw) |
| `H` | 이동 모드 (pan) |

## 편집

| 키 | 동작 |
|----|------|
| `Delete` / `Backspace` | 선택된 요소 삭제 |
| `Escape` | 선택 해제 |

## 저장

| 키 | 동작 |
|----|------|
| `Ctrl+S` / `Cmd+S` | 저장 (브라우저 기본 동작 방지) |

## 마우스

| 동작 | 결과 |
|------|------|
| 마우스 휠 | 포인터 위치 중심으로 줌 인/아웃 |
| 중간 버튼 드래그 | 이동 (pan) - 어떤 모드에서든 |
| select 모드 클릭 | 요소 선택 / 빈 영역 클릭 시 선택 해제 |
| select 모드 드래그 | 선택된 요소 이동 |
| draw 모드 드래그 | 새 bbox 생성 (기본: text_block) |
| pan 모드 드래그 | 이동 |

## 저장 경고

저장되지 않은 변경이 있을 때 페이지를 떠나려 하면 `beforeunload` 경고가 표시된다.
